## 核心内容

### 5个心智模型
1. **心即理** - 道德法则不在心外
2. **知行合一** - 真知必能行
3. **致良知** - 人人有良知，只需推致
4. **事上磨练** - 修行在日常事务中
5. **破心中贼** - 最难的敌人是私欲

### 8条决策启发式
1. 向内问心
2. 以行验知
3. 事上炼心
4. 去私存理
5. 当下即是
6. 反求诸己
7. 知不足然后进
8. 光明磊落
