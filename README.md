# 王阳明视角Skill

王阳明心学思维框架与修身实践指南。

## 安装

将此目录链接到你的 AI 工具的 skills 目录：

```bash
# Claude Code
ln -s $(pwd) ~/.claude/skills/wangyangming-perspective

# Windsurf / Cursor
ln -s $(pwd) ~/.windsurf/skills/wangyangming-perspective
```

## 使用

### 显式触发词
- 「用王阳明视角」「王阳明会怎么看」「阳明视角」「致良知角度」
- 「心即理是什么」「知行合一怎么理解」「致良知的含义」

### 概念触发词
- 「良知」「心安吗」「知道但做不到」「事上磨练」「心中贼」

### 情境触发
- 面临道德困境、价值选择
- 纠结、反复权衡利弊却无法决定
- 知道应该做但迟迟未行动
- 想修身、想静心、想提升心性

## 功能

### 思维顾问
- 用王阳明视角分析问题、审视决策
- 5个核心心智模型：心即理、知行合一、致良知、事上磨练、此心不动
- 8条决策启发式

### 修身实践
- 静坐省察：每日静坐10-15分钟，觉察心有不安处
- 事上磨练：把困难当作修行道场
- 致良知三问：遇抉择时的三个问题
- 克治私欲：觉察并克制私欲萌芽
- 立志日省：每日反省言行是否与志向一致
